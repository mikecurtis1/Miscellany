HOW TO ADD SLIDES FOR A SLIDE SHOW... 

1. Put a set of jpg files into "C:\kiosk\content\slides"

2. Rename all the jpg files as a batch with Windows Explorer

  see: http://support.microsoft.com/kb/320167
	
	a) Select all jpg files, press F2.
    b) Type "slides.jpg", and then press ENTER.
	c) open "C:\kiosk\app\js\appendslides.js" then... 
	d) change var "total" to the total number of jpg files in "C:\kiosk\content\slides"
